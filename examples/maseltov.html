Split


Client: SGI
Role:Level Design & Gameplay Programming

About

A Mobile platformer used to display differing cultures.
My work on this project focused solely around gameplay programming, making a variety of platforms(springboard, moving etc) to enhance the levels.
This was one of my first projects in Unity, and on mobile. One of the biggest hurdles for me was adapting to this engine engine and the principles of programming for mobile.
Though I started with the classic platforming elements, such as springboards and moving platforms, after a while I tried creating magnetic platforms and toying with permanence because of the players actions. Unfortunately I was unable to finish these within the scope of my internship.
My main focus during this project was the modularity of my code. Up till this point I hadn't worked as part of a large professional team, and as such I was not used to coding as part of a large mechanism. So the key goal I set myself in this project was to allow my code to be plugged in to anywhere relevant in the game with next to no impact on the other systems.
I would definitely like create something similar to this project with the mechanics of magnetism and permanence as I feel they could be unique and fun mechanics in a modern platformer.
