About

A choose your own adventure style dialogue engine. This was a small self managed project that had only one coder and artist. We we're given a script by the client and around a week per resource to get a working project up and running. One of the benefits to using unity for this project was that the output of the project worked well on all devices.
The main constraint on this project was time, as I only got to spend roughly a week on the engine I had to focus on creating an adaptable system that could achieve all of the tasks set out in the clients script. To save time I also used an asset from the Unity asset store which unfortunately caused issues with the content input of the project. However this did not effect the final build.
I wanted to use this project to teach myself 'Delegates' as well as XML Serialisation, both very useful tools to have in C# development. Though this project unintentionally also teached me some minor managerial skills due to its self managed nature.
I wanted to make the system as user friendly as possible for future projects. The goal being that next to no coder pressence was needed to re-use the engine. This meant that the system could be used to rapidly prototype future projects, and hopefully expanded into a more powerful system if given the time a resources.
If I could redo this project I would remove the asset that was causing problems and rewrite the related systems accordingly. As well as adding a few features to give more varying effects on the way the text is presented. I would also like to add in the functionality for multiple speakers, though this can be achieved with the current system it is not designed with that in mind and would have issues accomplishing it.
