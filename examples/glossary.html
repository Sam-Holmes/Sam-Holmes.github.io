THE GLOSSARY APP

Client: Totem Learning
Role: Development

About

The Glossary app was built using Javascript and PhoneGap, with the aim being to create a multiplatform companion app for projects at totem. The app draws from a Content Management System and displays terms, their definitions and a relating graphic for a user to view. I was tasked with adding new features and implementing the updated design to this project. As well as cloning and rebranding the project.
At the beginning of this project I only had a very basic understand of Javascript, and as such quickly needed to apply my knowledge from other languages to the Javascript syntax. As a result this project served in teaching me the Javascript fundamentals along with improving my existing coding habits.
I used this project mainly as a tool to learn Javascript, however in doing so I also exposed myself to a fair amount of HTML and with the use of PhoneGap I was exposed to various challenges faced with android/iOS development. The product needed to work on multiple platforms, therefore I needed to avoid the use of any platform specific APIs, hence the use of PhoneGap.
If I was to revisit the project, I would split the product into platform specific apps. Though it was simpler to use PhoneGap and have the products work on multiplatforms, I feel a better result could be achieved by using the native development kits.
